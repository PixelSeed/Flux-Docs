    使用教程¶

    本文档聚焦当前实现里最常用的几条工作路径。

    1. 打开工具盒¶

    from floating_toolbox import show_toolbox
show_toolbox()

    这会进入默认的工具盒启动流程，并初始化面板、标记菜单与相关运行时状态。

    2. 三种界面模式¶

    Idle Panel¶

    • 适合常驻，不遮挡主要工作区域。
    • 适合把工具盒当成低干扰入口使用。

    Quick Panel¶

    • 适合放高频短操作。
    • 适合依赖收藏、常用工具和快速点击的工作流。

    Full Panel¶

    • 提供完整搜索、分类浏览和工具管理能力。
    • 更适合查找工具、批量整理和执行复杂入口。

    你也可以直接打开完整面板：

    from floating_toolbox import show_full_panel
show_full_panel()

    3. Full Panel 快捷键切换¶

    工具盒支持通过设置页保存一个“界面1/3切换快捷键”。

    当前行为是：

    • 如果当前已在 Full Panel，再次触发会切回 Idle Panel
    • 否则切到 Full Panel
    • 切入 Full Panel 时会自动聚焦搜索框

    4. 搜索与执行工具¶

    常见操作方式：

    1. 进入 Full Panel
    2. 在搜索框输入关键词
    3. 通过分类或子分类缩小范围
    4. 单击工具项执行

    如果某个工具是 UI 型工具，正常行为应是打开对应界面，而不是直接执行破坏性操作。

    5. 标记菜单¶

    工具盒内置标记菜单体系，适合视口内快速操作。

    推荐流程：

    1. 打开标记菜单编辑器
    2. 配置径向区和线性区
    3. 保存模板
    4. 激活模板并在视口中触发

    启动工具盒时，系统会尝试恢复上次使用的模板；如果没有，就会尝试加载默认模板。

    6. 设置页¶

    设置页目前包含几类主要内容：

    • 常规
    • 分类栏
    • 工具路径
    • AI
    • 数据看板

    常见可配置项包括：

    • 界面语言
    • Full Panel 缓存
    • Full Panel 快速切换动画
    • I1/I3 切换快捷键
    • 额外工具路径
    • AI 提供商、模型、Base URL、API Key

    详细说明见 配置说明。

    7. AI 任务调度¶

    如果你已经开启 AI 功能，可以打开 AI 任务对话框：

    from floating_toolbox import show_ai_task_dialog
show_ai_task_dialog()

    为了便于试验编排链路，也可以先同步一批演示工具：

    from floating_toolbox import seed_ai_demo_tools
seed_ai_demo_tools(overwrite=False)

    8. 关闭工具盒¶

    from floating_toolbox import close_toolbox
close_toolbox()

    关闭时系统会尝试刷新待保存状态并清理标记菜单资源。
