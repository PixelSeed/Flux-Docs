    使用教程¶

    本章节将详细介绍 Maya Floating Toolbox 的核心功能和操作流程。

    界面模式¶

    工具提供两种主要的界面模式，以适应不同的工作流：

    1. 浮动面板 (Full Panel)¶

    这是默认的展开模式，提供了最完整的功能： - 搜索功能：通过顶部的搜索栏快速查找工具。 - 分类浏览：左侧（或上方）展示工具分类，支持多级目录。 - 自定义头像：支持个性化头像显示。 - 热重载：右键头像可以一键重载所有脚本插件。

    2. 停靠面板 (Docked Panel)¶

    如果您希望工具箱始终可见且不遮挡视口，可以将其停靠在 Maya 侧边栏： - 点击面板右上角的 Dock 图标进行停靠。 - 再次点击 Undock 可恢复为浮动窗口。

    标记菜单 (Marking Menu)¶

    标记菜单是本工具的核心加速器。

    配置您的菜单¶

    1. 打开 Marking Menu Editor。
    2. 径向区域：拖拽工具到 8 个槽位中。
    3. 线性区域：在下方列表添加额外的常用命令。
    4. 保存模版：您可以为不同的环节（如角色绑定、特效）保存不同的模版。

    在视图中使用¶

    在 Maya 视图中，按住 Ctrl + Alt + 中键 即可呼出当前激活的标记菜单。

    工具管理¶

    添加新工具¶

    • 在面板空白处右键或通过菜单选择 "新建工具"。
    • 输入工具名称、图标、分类以及 Python 代码。
    • 支持直接在编辑器中测试运行。

    编辑/删除¶

    • 右键点击任何现有工具图标。
    • 您可以随时修改代码或将其移动到其他分类。
