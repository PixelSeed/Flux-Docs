    安装指南¶

    本章节将引导您完成 Maya Floating Toolbox 的安装。

    环境要求¶

    • Maya 版本：Maya 2023+ (推荐 Maya 2026)
    • Python 版本：Python 3.9+
    • 依赖库：Qt.py

    安装步骤¶

    1. 下载工具¶

    将项目代码克隆或下载到您的本地目录。

    2. 配置 Maya 模块¶

    在您的 Maya 脚本目录（例如 Documents/maya/2026/modules）中创建一个 .mod 文件，内容如下：

    + MayaFloatingToolbox 1.0 <您的代码路径>
scripts: .

    3. 安装依赖¶

    确保您的 Maya Python 环境安装了必要的包：

    mayapy -m pip install Qt.py

    4. 启动工具¶

    在 Maya 脚本编辑器中运行以下 Python 代码：

    from floating_toolbox import main
main.show()
