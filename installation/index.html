    安装指南¶

    本文档面向第一次安装 Maya Floating Toolbox 的用户。

    环境要求¶

    • Maya 2017 或更新版本
    • 具备在 Maya Script Editor 中运行 Python 的能力

    安装步骤¶

    第一步：解压安装包¶

    将下载的压缩包解压到你希望长期存放的目录，例如：

    C:\Users\你的用户名\Documents\MayaTool

    解压后你会看到一个包含 floating_toolbox 子目录的文件夹。

    注意：安装后不要移动或删除这个文件夹，Maya 通过注册的路径直接读取文件。

    第二步：运行安装脚本¶

    有三种方式可以运行安装脚本，任选其一：

    方式 A（推荐）：拖拽安装¶

    将解压目录中的 floating_toolbox/install.py 文件直接拖入 Maya 的视窗（Viewport），松开即自动执行安装。

    方式 B：Script Editor 执行¶

    打开 Maya 的 Script Editor，切到 Python 标签，执行以下代码（替换路径为你的实际解压路径）：

    exec(open(r"C:\Users\你的用户名\Documents\MayaTool\floating_toolbox\install.py", encoding="utf-8").read())

    方式 C：双击执行¶

    在文件浏览器中找到 floating_toolbox/install.py，双击运行（需要系统已关联 Python 解释器）。

    安装脚本会自动完成以下操作：

    • 检测本机已安装的 Maya 版本
    • 将 floating_toolbox.mod 写入 Documents/maya/modules/ 目录（所有 Maya 版本共享）
    • 在 Maya 中创建 FloatingToolbox Shelf 和 Toolbox 快捷按钮

    第三步：重启 Maya¶

    安装完成后重启 Maya，确保模块路径和 Shelf 按钮正常加载。

    启动工具盒¶

    重启后，点击 Maya 工具栏中的 FloatingToolbox Shelf 下的 Toolbox 按钮即可打开。

    也可以在 Script Editor 中执行：

    from floating_toolbox import show_toolbox
show_toolbox()

    安装后验证¶

    1. 工具盒主窗口可以正常打开
    2. Maya 顶部工具栏出现了 FloatingToolbox Shelf
    3. 设置页可以正常打开
    4. 如果依赖快捷键，在设置中保存后能正常生效

    手动验证模块文件¶

    安装脚本会将 .mod 文件写到共享 modules 目录，路径类似：

    Documents/maya/modules/floating_toolbox.mod

    如果工具盒无法加载，确认该文件是否存在，并检查文件内容中的路径是否指向你的解压目录。

    数据路径¶

    用户配置和数据默认保存在：

    Documents/maya/prefs/floating_toolbox_shared

    升级¶

    下载新版本后，解压覆盖原目录，然后在 Maya 中重新执行一次 floating_toolbox/install.py 即可完成升级。

    卸载¶

    在 Maya Script Editor 中执行：

    exec(open(r"你的解压路径\floating_toolbox\uninstall.py", encoding="utf-8").read())

    卸载脚本默认只移除模块注册和 Shelf 入口，不会删除用户数据。只有在卸载时明确确认，才会清除配置和缓存。

    遇到问题？¶

    • 确认 Documents/maya/modules/floating_toolbox.mod 文件存在
    • 确认文件内容中的路径指向实际的解压目录
    • 如仍无法解决，请通过下载页面的联系方式反馈
